This plugin adds a Font Family Dropdown to Visual Editor.

== Description ==
This is a fairly basic plugin and adds a Font family dropdown for selecting fonts.

Visit <a href="http://www.priteshgupta.com/plugins/font-family-dropdown" target="_blank">Add Font Family Dropdown to Visual Editor Plugin Homepage</a> for more information.

<strong>NOTE: This plugin also removes the "Paste as Plain Text", "Paste from Word" and "Remove formatting" button for neat layout of Visual Editor, though this is optional.</strong>

<strong><a href="http://www.priteshgupta.com/contact" target="_blank">Contact Me</a> if you will like to see any other feature on this plugin.</strong>

== Installation ==
1. Download the latest version
2. Extract it in the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

Visit <a href="http://www.priteshgupta.com/plugins/font-family-dropdown" target="_blank">Add Font Family Dropdown to Visual Editor Plugin Homepage</a> for more information.
